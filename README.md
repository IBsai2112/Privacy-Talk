# Privacy-Talk

**Privacy-Talk** is a **real-time chat application** built using the **MERN stack** (MongoDB, Express, React, Node.js).  
It offers secure user authentication, group chats, live messaging via **Socket.IO**, and a responsive UI styled with **Tailwind CSS**. :contentReference[oaicite:1]{index=1}

## 🚀 Features

✔ User authentication (sign up / login)  
✔ Real-time one-to-one and group chats  
✔ Live messaging with Socket.IO  
✔ Global state management using Redux  
✔ Fully responsive interface (mobile + desktop)  
✔ Clean UI with Tailwind CSS :contentReference[oaicite:2]{index=2}

## 🛠 Installation (Local)

### 1. Clone the Repo
```bash
git clone https://github.com/IBsai2112/Privacy-Talk.git
cd Privacy-Talk
```
```bash
cd backend
npm install
```

## Create .env with your environment variables (e.g., MONGO_URI, JWT_SECRET)
```bash
npm start
```

```bash
cd ../frontend
npm install
npm start
```

MONGO_URI=your_mongo_connection_string
JWT_SECRET=your_secret_key
